About NubeSecure

NubeSecure is a cloud security product and service provider. Founder Brandon Chen has been a hands-on engineer for over 16 years. He became passionate about cloud security after leading the development and delivery of several security products. NubeSecure is actively developing a light-weight product suite for application and data security with the goal to provide premium services in the cloud space.
